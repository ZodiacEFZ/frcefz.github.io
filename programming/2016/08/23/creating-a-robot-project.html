原文：Creating a robot project

创建项目

右键点击 Project Explorer 中的空白区域，选择 New - Project...

选择项目类型

展开 WPILib Robot C++ Development / WPILib Robot Java Development 文件夹，选择合适的项目类型 (Robot C++ Project 或 Robot Java Project)。这里只能看到安装过的项目类型。

选择项目骨架

在文本框中给项目命名，然后选择 Command-Based Robot 单选框。

在 Project Explorer 中浏览项目骨架

注意当前浏览的可能是其他项目。请选择一个目录下有 commands 和 subsystems 的项目。
